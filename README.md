# Entunelador
El puerto debe de ser con un numero entero por ejemplo si el localhost es http://localhost:8080 el puerto que pondremos sera 8080
# Instalacion
```
apt install git -y
apt install python3 -y
cd $HOME
git clone https://github.com/Euronymou5/Entunelador
cd Entunelador
python3 entun.py
```
